JavaScript Tests
================

![Exmaple](http://ibin.co/24s8OoIZeuDs "Exmaple of usage")

Examples of various JavaScript unit test frameworks.

Installation
=============

Installation is as easy as:
``npm install``

Running
=======

Jasmine Tests:
``npm run jasmine``

Mocha Tests:
``npm run mocha``

QUnit Tests:
``npm run qunit``
